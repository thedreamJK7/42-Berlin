## 👋 About

This repository is the central hub for my journey through the **42 Berlin Common Core** curriculum.

42 is a peer-to-peer coding school with no teachers, no lectures — just projects, peer reviews, and deep problem solving. The curriculum covers everything from low-level C programming to full-stack web development.

📍 Current status: **Rank 5**

## 🏆 Project Highlights

### 🐚 Minishell — [repo](https://github.com/thedreamjk7/minishell)
> Rank 3 | Group Project

Recreation of a Unix shell similar to bash.
- Custom lexer/parser and AST design
- Process control (`fork`, `execve`)
- Pipes, redirections, and signal handling (`Ctrl-C`, `Ctrl-D`, `Ctrl-\`)

---

### 🍝 Philosophers — [repo](https://github.com/thedreamjk7/philosophers_42)
> Rank 3 | Concurrency

Classic Dining Philosophers problem using threads.
- Mutexes & synchronization
- Deadlock prevention
- CPU usage optimization

---

### 🎨 so_long — [repo](https://github.com/thedreamjk7/so_long)
> Rank 2 | Graphics

Small 2D game built with MiniLibX.
- Window management and texture rendering
- Key-hook event handling
- Game loop and map parsing

---

### 📊 push_swap — [repo](https://github.com/thedreamjk7/push_swap)
> Rank 2 | Algorithms

Sort a stack of numbers using two stacks and a minimal set of operations.
- Complexity analysis (Big O)
- Stack manipulation
- Bitwise techniques

---

### 🌅 miniRT — [repo](https://github.com/thedreamJK7/miniRT_42Berlin)
> Rank 4 | Group Project | Ray Tracing

A ray tracer built from scratch in C using MiniLibX. Renders 3D scenes with lighting, shadows, and geometric objects.
- Implemented ray-object intersection (spheres, planes, cylinders)
- Phong lighting model (ambient, diffuse, specular)
- Camera, viewport, and scene parsing from `.rt` files
- Multi-object scenes with shadows and reflections

---

### 🧪 minitalk — [repo](https://github.com/thedreamJK7/minitalk)
> Rank 2 | Unix Signals

Client/server communication using only UNIX signals.
- Deep understanding of `SIGUSR1`, `SIGUSR2`
- Bitwise encoding/decoding of characters
- Signal handling with `sigaction`
